# Proyecto de Graficación

*COMPILACION:*
Abrir Code::Blocks, abrir proyecto en `src/`, Build → Build and Run.

*EJECUCIÓN:*
Ejecutar el archivo `main.cpp`. Seleccionar la figura a dibujar:
- Línea directa
- Línea DDA
- Círculo incremental
- Círculo punto medio
- Elipse punto medio

*PARÁMETROS:*
- Línea: xi, yi, xf, yf
- Círculo: xc, yc, r
- Elipse: xc, yc, rx, ry

*EJEMPLOS:*
- Línea directa: (2,1) a (5,9)
- Círculo punto medio: centro (0,0), radio 10
